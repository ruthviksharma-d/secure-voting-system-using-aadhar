Fingerprint Registration Process

File Naming Guidelines

Your fingerprint file must be named exactly as registered in our system:

firstname_fingerprint.extension

Examples:

  • john_fingerprint.wsq
  • sarah_fingerprint.jpg
  • michael_fingerprint.png
  • emily_fingerprint.jpeg
Supported Formats

We accept the following fingerprint file formats:

.WSQ
Wavelet Scalar Quantization
.JPG
JPEG Image
.PNG
Portable Network Graphics
.JPEG
JPEG Image
Important: Your file name must exactly match what's registered in our voter database. Case sensitivity and spelling matter!
